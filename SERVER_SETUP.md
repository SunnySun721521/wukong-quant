# 服务器配置指南

## 情况说明
您的手机使用5G网络，与电脑不在同一个局域网，因此无法通过局域网IP地址访问电脑上的服务器。

## 解决方案

### 方案1：使用ngrok内网穿透（推荐）

#### 1. 下载ngrok
访问 https://ngrok.com/download 下载适合您操作系统的ngrok版本。

#### 2. 运行ngrok
在命令行中运行：
```bash
ngrok http 5001
```

#### 3. 获取公网地址
ngrok会显示一个公网地址，例如：
```
Forwarding  https://abc123.ngrok.io -> http://localhost:5001
```

#### 4. 在小程序中配置
1. 打开小程序
2. 点击首页右上角的"⚙ 设置"按钮
3. 输入ngrok提供的地址（例如：`https://abc123.ngrok.io`）
4. 点击"测试连接"按钮
5. 点击"保存"按钮

### 方案2：部署到云服务器

#### 1. 购买云服务器
- 阿里云：https://www.aliyun.com
- 腾讯云：https://cloud.tencent.com
- 华为云：https://www.huaweicloud.com

#### 2. 上传代码
将后端代码上传到云服务器。

#### 3. 运行服务
在云服务器上运行后端服务。

#### 4. 配置小程序
在手机上输入云服务器的公网IP地址，例如：`http://123.45.67.89:5001`

### 方案3：使用微信小程序云开发

#### 1. 开通云开发
在微信开发者工具中点击"云开发"，开通云开发环境。

#### 2. 创建云函数
将后端逻辑改写为云函数。

#### 3. 无需配置服务器地址
云开发环境会自动处理网络请求。

## 推荐方案

如果您只是想快速测试，推荐使用**ngrok**，因为：
- 免费版本足够使用
- 配置简单
- 不需要购买云服务器
- 适合开发和测试

## 注意事项

1. ngrok免费版地址会变化，每次重启ngrok都会生成新的地址
2. 如果需要稳定的公网地址，建议购买云服务器
3. 生产环境建议使用云服务器或云开发
